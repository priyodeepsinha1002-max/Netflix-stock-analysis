# Netflix (NFLX) Equity Analytics & Market Volatility Engine

A professional time-series financial analytics project designed to clean, transform, and evaluate the historical equity lifecycle of Netflix Inc. (NFLX). This project extracts actionable market insights by mapping macro trend indicators, examining trading volume structures, and quantifying daily asset risk distribution models.

## 🚀 Key Analytical Features
* **Advanced Data Ingestion Pipeline:** Automatic extraction and removal of text-based ticker identifiers, automated row-level data type conversions, and time-series datetime index structuring.
* **Multi-Interval Volume Diagnostics:** Deep-dive analysis profiling trading volumes across daily, monthly, and yearly horizons to identify institutional rebalancing cycles.
* **Dual-Horizon Trend Filtering:** Implementation of 50-Day (Short-Term) and 200-Day (Long-Term) Simple Moving Averages to evaluate macro momentum signals.
* **Statistical Risk Engine:** Isolation of daily asset percentage returns to evaluate market volatility, central tendency trends, and leptokurtic distribution risks.
* **Compounded Growth Engineering:** Structural modeling of cumulative long-term investment performance via logarithmic time-series mapping.

## 🛠️ Technical Stack
* **Language:** Python
* **Data Manipulation:** Pandas, NumPy
* **Data Visualization:** Matplotlib, Seaborn
* **Environment:** Jupyter Notebook / Google Colab

## 📈 Core Financial Insights
1. **High Peak Centrality:** The daily return distribution displays a highly concentrated peak at 0%, proving that standard daily sessions are fundamentally balanced and stable.
2. **Asymmetric Fat-Tail Risks:** Rare macroeconomic and corporate developments drive large outlier sessions that exist as fat-tails, requiring robust portfolio hedging strategies.
3. **Logarithmic Scaling:** Compounded returns highlight exponential enterprise scale, demonstrating how sustained long-term business model adaptation creates geometric compounding value over decades.
